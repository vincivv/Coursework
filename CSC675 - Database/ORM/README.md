# ORM Reference Project

This project is a template for the ORM project in our database class.

Its purpose is to give you a simple starting point so you can study the code, reuse parts of it, and build your own project on top of it.

## How to use this project

- Read the TODO messages carefully
- Use it as a baseline for your own project
- Reuse ideas or structure where helpful
- Extend it based on your own requirements

## Important

This project is meant for learning. Do not copy it blindly. Make sure you understand any code you reuse.

## Goal

The main goal is to help you get started faster and give you a concrete example of how one possible ORM project can be organized.

## Notes

You are free to modify this project for your own work in the class.
